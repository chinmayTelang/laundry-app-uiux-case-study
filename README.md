# laundry-app-uiux-case-study
This repository contains my UI/UX design work for a Laundry App, created using Figma. It includes the complete design process from initial sketches and wireframes to high-fidelity UI, interactive prototypes, and reusable component systems. This project reflects my learning and practical implementation of modern UI/UX design principles.
# 🧺 Laundry App UI/UX Case Study

## 📌 Overview

This repository showcases my complete UI/UX design work for a Laundry Service Application created using Figma. It includes the entire design process from research and ideation to high-fidelity UI, prototyping, and design systems.

---

## 🎯 Objective

To design a user-friendly and efficient laundry service application that simplifies booking, tracking, and managing laundry services.
